WVS—漏洞扫描的使用

发表于 2019-06-05 更新于 2026-06-24 分类于 web学习

只是写一下关于漏洞扫描的使用，毕竟这个软件主要就是漏扫。

AWVS 11

添加描述，设置扫描速度

如果需要登陆，打开login，可在里面输入用户名密码

仪表框显示总体

可在scans打开扫描的网址看详细

AWVS 10.5

  • Website URL：输入要扫描的网址
  • Filename：选中后，可导入内置的爬虫结果进行漏洞扫描

  • Scanning profile：设置侧重扫描类型，一般默认
  • Scan settings：扫描器设置，一般默认
  • Show advanced options in the scan wizard：显示高级扫描选项设置

  • 扫描的网站信息
  • 目标的程序语言

  • 使用预先的登陆序列，没有直接默认
  • 输入用户名，密码。可随缘登陆某些界面
