CAD2014的安装，破解

发表于 2019-06-27 更新于 2026-06-24 分类于 专业学习

更改可选择解压目标文件夹，然后确定直接下一步进行自动解压安装，建议在选择的目录下建立CAD2014文件夹，然后选择此文件夹安装。注：安装的电脑最少需有两个盘，不然会导致部分功能无法使用或者安装失败。

解压安装进行中。。。。

点击安装，进行下一步。

国家中国不用更改，下面选择我接受，然后下一步。

上面默认，下面序列号可填666-69696969密钥001F1。当然，也可以百度一些别的。然后下一步。

安装路径可选创建在别的盘下的CAD2014文件夹，也可选别的，然后下一步。

等待安装。。。。。

安装到这里就差不多完成了，然后打开软件进行激活。

下载完成之后会有两个图标或者三个图标，选择红色那个CAD图标打开。

打开这个图标，进行激活操作

勾选我同意声明，然后点我同意，进行下一步。

会出现许可加载框，直接关闭。

直接点击激活（用注册机来破解），如果点击试用，则只能使用30天，30天后需要激活码进行激活。

激活后，如果跟上图不一样，直接关闭，重新打开CAD，按照步骤点击激活即可。选中我具有激活码，然后打开注册机。

两个，分别是32位和64位的，电脑系统位数可“右键我的电脑-属性”查看。然后选中对应的注册机打开。（建议右键注册机使用管理员权限打开）

首先选中patch按钮进行破解，然后点击弹出框的确定。

然后把第三步出来的激活码直接复制到激活页面里面即可，然后点击下一步，就成功破解。

本地搭建DVWA的部分错误

发表于 2019-06-15 更新于 2026-06-24 分类于 web学习

额。。。。。因为太菜了，搭建个本地平台都是出各种错误。。。。我就把错误做了总结，顺便从网上找了各种解决的办法。。。

1.创建数据库,出现错误

Could not connect to the MySQL service.
Please check the config file.

无法连接到MySQL服务。请检查配置文件。

打开phpmyadmin,用登录这个的密码,打开WWW下DVWA的config配置文件,更改
$_DVWA[ ‘db_password’ ] = ‘p@ssw0rd‘;  密码改为登录phpmyadmin的密码即可(默认为root).然后登录DVWA默认admin/password

2.系统错误,找不到配置文件

DVWA System error - config file not found. Copy config/config.inc.php.dist to config/config.inc.php and configure to your environment.

请复制config目录下的config.inc.php.dist文件 到 config目录下的config.inc.php文件，并配置你的环境。

在config下找到 config.inc.php.dist文件 ,去掉后面的.dist后缀.

3.本地burp suite抓包失败

wdnmd，代理设了，bp开了，抓不到包，有什么好说的。。。。

局域网——高级——删除不使用代理服务器框里的所有字——OK

或者都弄完了还是不能抓（反正我是。。呜呜呜）。。。。。。。。。下面的步骤

在cmd找到本机IP，然后。。。。

把访问DVWA的网址换成本地IP，直接访问就可以，在进行抓包。。。嗯。。。成功！

终于可以愉快的学习了！！

4.配置出现的错误

问题：PHP function allow_url_include: Disabled
处理：编缉/etc/php.ini将allow_url_include值由Off改为On

问题：PHP module gd: Missing
处理：yum install -y php-gd

问题：reCAPTCHA key: Missing
处理：编缉dvwa/config/config.inc.php
配置$_DVWA[ ‘recaptcha_public_key’ ] = ‘6LdK7xITAAzzAAJQTfL7fu6I-0aPl8KHHieAT_yJg’;
配置$_DVWA[ ‘recaptcha_private_key’ ] = ‘6LdK7xITAzzAAL_uw9YXVUOPoIHPZLfw2K1n5NVQ’;

问题：Unable to connect to the database
处理：编缉dvwa/config/config.inc.php，将$_DVWA[ ‘db_password’ ]的值改成自己设的数据库root账号的密码

部分CAD常用的命令使用

发表于 2019-05-16 更新于 2026-06-24 分类于 专业学习

直线：l 直接输入，直接画。

创建块：b 先选中全部图形，输入“B”创建块，在里面点击拾取点，任意点击图形位置，选择全部图像，进行图形块的保存。

偏移：o 先选中图形，然后指定偏移距离，在偏移的方向点击即可。

修剪：tr 输入“TR”，按两下空格，选中修剪的线段即可。

旋转：ro 输入“ro”，输入旋转的度数“90°”或“-90°”。

复制：co 输入“co”，选择复制的位置即可。

删除：e 选中图像，输入“e”，回车即可。

移动：m 选中图像，输入“m”，指定移动方向。

填充：h 输入“H”，空格选择需要的填充类型，选中填充的区域，进行填充，若填充存在超越边界，可选择删除边界来调整填充。

射线：ray 输入“ray”选中一个方向单击。

裁剪：clip 首先把图保存成块，输入“clip”选中要裁剪的范围，选择多段线，进行矩形等范围裁剪。裁剪消失的图像可恢复。

阵列：ar 选中图像，输入“ar”，在打开的设置框里面设置需要阵列的行列数，设置阵列行列的距离，确定。

镜像：mi 选中图像，，输入“mi”，选中图像上一点，再选中对比点，进行镜像，镜像后可选择删除源图像或保留。

矩形：rec 直接输入命令“rec”，用XXX@，XXX来确定矩形大小。

多线：ml 输入“ml”，使用多线命令，可输入提示字母来调整多线的比例，也可输入mlstyle进去设置界面设置。

测量：di  输入选中一点，再选中另一点，直接测量。

F10——极轴追踪
