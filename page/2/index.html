先建立一个球体，按快捷键M打开材质编辑器。

可自己调节喜欢的颜色，下面的反射光也可自己调整。

建筑装饰设计笔记

发表于 2019-12-28 更新于 2026-06-24 分类于 WordPress迁移

——情景一

建筑装饰设计概念：

建筑装饰设计即通过物质技术手段和艺术手段，为满足人们生产、生活的物质需求和精神需求而进行的建筑室内外空间环境的创造设计活动。

建筑装饰设计的发展：

1.原始社会——2.夏、商、西周和春秋战国时期——3.秦汉时期——4.魏晋南北朝——5.隋唐五代——6.宋代——7.明清时期（明清时代的家具一般以清代乾隆时为分界线，此前的家具称“明式家具”，此后的家具称“清式家具”）

建筑装饰设计发展趋势：

1.相对独立性增强。2.继续多元化发展道路。3.艺术与技术结合得更加紧密。4.动态发展趋势。5.可持续发展趋势。

建筑装饰设计的目的：

两个方面：1.物质需求方面。使空间功能更加合理，并利用现代建筑技术，改善室内物理环境，使人们的生产、生活更加安全、舒适、高效。2.精神需求方面。塑造具有强烈精神感受的视觉空间环境，以求视觉感官和精神上的审美效果，使人们在有限的空间内获得无限的精神感受。

建筑装饰设计的内容：

1.室外环境设计。2.室内环境设计。3.家具设计。4.效果设计。5.技术要素设计。6.材料的选择。

——情景二

建筑装饰设计的构成要素：

1.空间要素。2.光影要素。3.色彩要素。4.材料要素。5.界面要素。6.陈设要素。

建筑装饰的设计原则：

1.经济适用原则。2.物质文明与精神文明并进原则。3.时代感与历史文脉并重原则。4.个性化原则。5.动态发展原则。

⭐建筑装饰设计方法：

1.立意创新。2.细部入手、逐步深入。3.树立整体观念。

⭐人体工程学概念：

人体工程学，又称”人类工程学“”人体工学“和”人类工学“等。它是一门研究人与机器及环境关系的技术科学，是以人的心理学、解剖学和生物学为基础，综合多种学科研究人与环境的各种关系，使得生产器具、生活器具、工作环境、生活环境等与人体功能相适应的一门综合性学科。

人体尺度：

1.静态尺度。2.动态尺度。

人体尺度测量方法：

1.形态尺度：形态测量是指对人体的基本尺度、体型、表面面积、体积和质量等所进行的测量，是以检测人体静止形态为主的一种测量方式。

2.运动测量：运动测量是指人体关节的活动范围和肢体活动空间的测量，如动作范围，动作过程，形体变化和皮肤变化等，其以人体静止形态测量为基础，是以检测人体的动作过程（如肢体活动范围的大小和操作动作的过程等）为主的一种测量方式。

——情景三

室内空间环境的概念：

人工环境的室内空间是人类劳动的产物，是相对于自然空间而言的，是人类有序生活组织所需要的物质产品。

⭐室内空间的构成：

基面、顶面、垂直面。

⭐室内空间的类型：

1.固定空间与可变空间。2.动态空间与静态空间。3.开敞空间与封闭空间。4.虚拟空间。5.虚幻空间。6.凹入空间与外凸空间。

空间分隔方式：

空间分隔的方式主要有绝对分隔、局部分隔、象征性分隔、弹性分隔四种。

公共性——半公共性——半私密性——私密性

开敞性——半开敞性——半封闭性——封闭性

室 外——半 室 外——半 室 内——室 内

人在建筑室外空间环境的行为感受：

两个方面：建筑室外空间是一个过度空间。建筑室外空间也是一个共享空间。

室外空间环境设计原则：

人本化与自然化的设计原则：1.首先，对于建筑室外空间环境而言，物质功能是最基本的要求。2.其次，行为功能和文化内涵是建筑室外环境的精神功能。3.最后，寻求与自然的平衡是建筑室外空间环境设计的一个重要准则。

——情景四

各类界面的共同要求：

1.耐久性及使用期限。2.耐燃性及防火性能。3.无毒。4.无害。5.易于制作、安装和施工，便于更新。6.满足隔热、保暖、隔声、防火、防水要求。7.满足装饰美观性要求。8.满足相应的经济要求。

材料的分类：

1.点状材料。2.线状材料。3.面状材料。4.块料材料。5.结合材料。6.自然材料。7.人工材料。

室内空间界面材料的选用：

1.适合室内使用空间的功能性质。2.适合建筑装饰的相应部位。3.符合更新、时尚发展的需要。

⭐室内空间界面设计的原则：

1.装饰与室内空间各界面特定要求相协调。2.与室内气氛相一致。3.室内空间界面处理上切忌过分突出。

⭐选择材料特征注意点：

1.要使材料特性与空间性格相吻合。2.要充分展示材料自身的内在美。3.注意材料质感与距离面积的关系。4.注意与使用要求的统一。5.注意用材的经济性。

⭐室内空间处理：

分割、切断、通透、裁剪、高差、凹凸、借景。

界面、家具、园林、陈设和灯饰是构成室内空间的五大基本要素。

⭐形状、色彩、质感是物体的三个视觉形态，它们各具有自己的特性。

室内空间要素的色彩组合常有：对称法、重复法、退晕法、换色法、对比法。

——情景五

⭐色彩三要素：色相、明度、纯度

间色概念：由两种原色调配色其他色彩混合而成色。

复色概念：由两种间接色调制而成的色。

原色概念：是指在颜色中不能分解的，并能调配出其他色彩的色。

色彩混合分为加色混色、减色混合和中性混合。

色彩与照明：

1.强光照射下，色彩会变淡，明度提高，纯度降低。

2.弱光照射下，色彩变模糊，色彩的明度、纯度都会降低。

色彩的作用：

色彩的生理作用。色彩的心理作用。色彩的物理作用。

色彩的心理作用：

直接心理作用。间接心理作用。

⭐色彩设计原则：

1.功能性原则。2.时空性原则。3.从属性原则。4.符合民族习惯和环境特点原则。5.色彩设计的沉着性原则。6.符合形式美原则。

⭐色彩的设计方法：

1.确定色调。2.合理配色。3.符合色彩构图法则。4.做好室内界面、家具、陈设的色彩选择和搭配。

——情景六

**照度：**光源在某一方向单位立体角内所发出的光通量叫作光源在该方向的发光强度（Lumminous Intensity），单位坎德拉（cd），被光照的某一面上其单位面积内所接收的光通量称为照度，其单位为勒克斯（lx）。

**亮度：**发光物体在给定方向的单位、投影面积的发光强度成为发光亮度，符号是L，单位是cd/m²。

**光色：**光色主要取决于光源的色温（K），光色还影响室内的气氛。

室内照明方式：

1.一般照明。2.局部照明。3.混合照明。4.成角照明。

灯具散光光量分类：

1.直接照明。2.半直接照明。3.漫射照明。4.半间接照明。5.间接照明。

室内照明设计原则：

1.实用性。2.安全性。3.经济性。4.舒适性。5.美观性。

室内照明作用与效果：

1.丰富室内空间、创造室内气氛。2.加强空间感和立体感。

建筑照明的形式：

1.窗帘照明。2.镶板式照明。3.花檐照明。4.发光顶棚照明。5.发光墙架照明。6.满天星照明。7.底面照明。8.光带照明与光梁照明。9.泛光照明。10.平衡照明。11.导轨照明。12.高低托架照明。13.环境照明。

——情景七

家具的分类：

1.按人体工程学法则分类。2.按使用功能分类。3.按制作材料分类（木制家具——竹藤家具——金属家具——塑料家具）4.按结构形式分类（板式家具——框式——拆装——折叠——充气——泡塑——浇铸）

⭐家具在室内空间环境中的作用：

1.分隔空间。2.组织空间、引导入流。3.营造空间氛围。4.改善空间形态。5.均衡空间构图。

家具的选用：

1.室内协调性的把握。2.空间性格的把握。3.通用性把握。4.对总体艺术效果的把握。

⭐家具的布置方法：

周边式布置、岛式布置、单边式布置、走道式布置

布置格局：行列式，分组式，曲尺式，回形式，中心式，对排式。

室内陈设的作用：

功能性陈设。装饰性陈设。

室内陈设搭配方法：色彩搭配法，色调配色法，对比配色法，风格配色法，形态配色法，风格搭配法

室内陈设品的选择从总体上讲要从文化内涵，陈设品造型，色彩质地选择等方面考虑。

⭐室内陈设品的布置：

1.墙面陈设。2.台面陈设。3.落地陈设。4.厨架陈设。5.悬挂陈设。

——情景八

⭐利用绿化组织室内空间、强化空间，主要表现在对空间进行分隔、引导、柔化、填充等方面。

室内绿化作用：

1.对环境的美化作用。2.组织空间。3.净化环境。4.陶冶情操。

室内绿化的布局：

1.点状布局。2.线状布局。3.面状布局。4.综合式布局。

室内绿化方法：

1.主点装饰。2.边角点缀。3.结合家具、陈设等布置绿化。4.沿窗布置绿化。5.垂直绿化。

植物的选择：

1.光照条件。2.使用者条件。3.形态上的要求。

小品的布置方法：

小品的设计选择应与环境设计总体效果统一考虑、不拘一格、精心布置、宁缺毋滥，力求做到巧妙得体、精致适当。根据室内小品精美、灵巧的特点，抓住小品的本质结合到造型中去，然后进行布局点缀，巧妙而得体，起到画龙点睛的作用。

——情景九

地面装饰设计形式：

1.木地面。2.块材地面。3.塑料地面。4.美术水磨石地面。5.马赛克地面。

墙面装饰设计原则：

1.充分考虑墙面与室内其他部位的统一。2.应该尽量考虑建筑的统一风格以及构件和空间的真实性。3.墙面材料比其他部分材料的耐久性略高。4.墙面质感、色彩，以及形状比例等要与室内气氛相协调。

本地搭建DVWA的部分错误

发表于 2019-06-15 更新于 2026-06-24 分类于 WordPress迁移

额。。。。。因为太菜了，搭建个本地平台都是出各种错误。。。。我就把错误做了总结，顺便从网上找了各种解决的办法。。。

1.创建数据库,出现错误

Could not connect to the MySQL service.
Please check the config file.

无法连接到MySQL服务。请检查配置文件。

打开phpmyadmin,用登录这个的密码,打开WWW下DVWA的config配置文件,更改
$_DVWA[ ‘db_password’ ] = ‘p@ssw0rd‘;  密码改为登录phpmyadmin的密码即可(默认为root).然后登录DVWA默认admin/password

2.系统错误,找不到配置文件

DVWA System error - config file not found. Copy config/config.inc.php.dist to config/config.inc.php and configure to your environment.

请复制config目录下的config.inc.php.dist文件 到 config目录下的config.inc.php文件，并配置你的环境。

在config下找到 config.inc.php.dist文件 ,去掉后面的.dist后缀.

3.本地burp suite抓包失败

wdnmd，代理设了，bp开了，抓不到包，有什么好说的。。。。

局域网——高级——删除不使用代理服务器框里的所有字——OK

或者都弄完了还是不能抓（反正我是。。呜呜呜）。。。。。。。。。下面的步骤

在cmd找到本机IP，然后。。。。

把访问DVWA的网址换成本地IP，直接访问就可以，在进行抓包。。。嗯。。。成功！

终于可以愉快的学习了！！

4.配置出现的错误

问题：PHP function allow_url_include: Disabled
处理：编缉/etc/php.ini将allow_url_include值由Off改为On

问题：PHP module gd: Missing
处理：yum install -y php-gd

问题：reCAPTCHA key: Missing
处理：编缉dvwa/config/config.inc.php
配置$_DVWA[ ‘recaptcha_public_key’ ] = ‘6LdK7xITAAzzAAJQTfL7fu6I-0aPl8KHHieAT_yJg’;
配置$_DVWA[ ‘recaptcha_private_key’ ] = ‘6LdK7xITAzzAAL_uw9YXVUOPoIHPZLfw2K1n5NVQ’;

问题：Unable to connect to the database
处理：编缉dvwa/config/config.inc.php，将$_DVWA[ ‘db_password’ ]的值改成自己设的数据库root账号的密码
